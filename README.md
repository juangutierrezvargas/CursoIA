# Proyecto Excel IA

## Autor
Juan Camilo Gutierrez Vargas

## Tema del proyecto
Análisis de ventas mensuales utilizando herramientas de inteligencia artificial en Excel.

## Pregunta del proyecto
¿En qué mes se proyectan mayores ventas según el análisis de Excel?

## Datos utilizados
Se utilizó una tabla de datos simulada con ventas mensuales para analizar tendencias y realizar una proyección.

## Herramientas de IA usadas en Excel
- Ideas (Analyze Data)
- Previsión de hoja (Forecast Sheet)
- Gráficos automáticos

## Resultados
El análisis muestra una tendencia de crecimiento en las ventas a lo largo del año. La previsión indica que los últimos meses presentan mayores ventas.
